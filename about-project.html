About This Project

Welcome! This page explains how I planned, designed, and built this website as part of my web development journey. Below, you’ll find a breakdown of the thought process, tools, and help I used to complete it.

Idea

The idea to create this website came from my desire to practice building real-world projects. I wanted a simple yet functional site to apply what I’ve learned so far and showcase my growing skills.

Plan

  • Start with a basic design and layout using only HTML and CSS.
  • Create an "About Me" section, a "Qualifications" section, and a "Skills" section to display relevant information.
  • Include a dedicated "Contact" section with my email and public phone number for communication.
  • Build an additional page, like this one, to document how the website was created.

Help

I received guidance from ChatGPT to:

  • Plan the structure of the website effectively.
  • Learn best practices for writing clean HTML and CSS.
  • Understand how to create links between pages and use proper styling.

Goal

The main goal of this project is to practice web development by building a complete website from scratch. This helps me improve my coding skills, build confidence, and prepare for learning JavaScript and other advanced technologies soon.

Future Improvements

  • Add JavaScript for dynamic interactions (e.g., form validation, animations).
  • Improve the design using advanced CSS techniques like Flexbox and Grid.
  • Eventually, host this website online to share my journey with others.
